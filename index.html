Uso excessivo de telas

Os dispositivos eletrônicos se tornaram muito na vida de muitas crianças pais e professores mas é de grande preucupação os perigos dessas tecnologias o uso excessivo dessas telas são prejudiciais para todos e devem ter seu uso controlado tanto para entretenimento quanto para atividades.

Como podemos diminuir o uso de telas

Uma forma é estabelecer limites de tempo para cada atividade online, seja trabalho ou lazer. Experimente usar alarmes ou lembretes para fazer pausas regulares.
